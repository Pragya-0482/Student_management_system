# Student Management System

A simple Python-based Student Management System that allows users to manage student records using file handling.

## Features

- Add student
- View students
- Search student by ID
- Update student details
- Delete student
- Count total students
- Save data permanently in a file

## Technologies Used

- Python 3
- Functions
- Lists
- Dictionaries
- Loops
- File Handling

## Project Files

```
Student-Management-System/
│
├── main.py
├── students.txt
└── README.md
```

## How to Run

1. Install Python 3.
2. Open the project folder in terminal.
3. Run:

```
python main.py
```

## Data Format

Student data is saved in `students.txt`:

```
ID,Name,Marks
```

Example:

```
1,Rahul,90
```

## Purpose

This project is created to practice basic Python programming concepts and file handling.
